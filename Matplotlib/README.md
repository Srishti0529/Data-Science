# Matplotlib

## 📌 What is Matplotlib?

Matplotlib is a **Python data visualization library** used to create **plots, charts, and graphs**. It helps visualize data in a clear and meaningful way.

It is commonly used with:

* NumPy
* Pandas
* Data Science projects
* Machine Learning

---

# Key Features

* Simple plotting interface
* Supports multiple chart types
* Highly customizable plots
* Works with NumPy & Pandas
* Export graphs as images
* Supports subplots
* Interactive visualization

---

# Types of Plots

## 1. Line Plot

Used to show trends over time.

Features:

* Displays continuous data
* Default plot type
* Useful for time series

---

## 2. Bar Chart

Used to compare categories.

Features:

* Vertical bars
* Horizontal bars supported
* Used for comparisons

---

## 3. Histogram

Used to show data distribution.

Features:

* Frequency distribution
* Continuous data
* Shows bins

---

## 4. Scatter Plot

Used to show relationship between two variables.

Features:

* Displays points
* Shows correlation
* Useful in ML

---

## 5. Pie Chart

Used to show percentage distribution.

Features:

* Circular chart
* Shows proportions
* Best for small categories
